A desktop GUI for Claude Code on Linux

Tabs, an embedded terminal, a Git panel, SFTP/FTP & SSH clients, and per-connection proxies — all on top of the claude CLI you already use.

⬇ Download for Linux ★ View on GitHub

Everything, in one window

Convenience of a GUI, the full power of the CLI.

🔑

No API key

Drives your local claude CLI, so it uses the same browser login and subscription — no key, no extra cost.

🗂️

Multiple tabs

Run several independent Claude sessions side by side, and drag tabs to reorder them.

💾

Persistent sessions

Close and reopen — your conversations are still there and resumable with full context.

🖥️

Embedded terminal

A real interactive claude in a PTY — slash commands, live permission prompts, plan mode.

⎇

Git panel

Stage, commit, push/pull/fetch, switch branches, view diffs and history — without leaving the app.

🌐

SFTP / FTP client

Browse remote files and upload, download, rename, create and delete — over SFTP, FTP or FTPS.

🔐

SSH tab

Save named servers and open a full interactive SSH shell right inside a tab.

🛡️

Per-connection proxy

Optional proxy per Git / SFTP / SSH connection — completely separate from the app proxy.

🎨

Rich rendering

Markdown with syntax-highlighted code, and a live view of each Read / Edit / Bash tool step.

🔒

Permission modes

Safe (ask), Accept edits, or Full access — chosen per tab, with a one-click Stop button.

📎

Drag & drop

Drop files to insert their paths into the prompt, or into the embedded terminal.

🌍

Themes & bilingual

Dark / light themes, adjustable font size, and a UI switchable between English and فارسی.

Five kinds of tab

Mix and match as many as you like.

💬 Chat 🖥️ Terminal ⎇ Git 🌐 SFTP / FTP 🔐 SSH

🛡️ Proxies that stay out of each other's way

Every Git, SFTP/FTP and SSH connection can carry its own optional proxy — HTTP CONNECT, SOCKS5 or SOCKS4.

It is fully independent of the app-wide proxy: if you fill it in, that connection uses it; if you leave it blank, it connects directly and is never routed through the app proxy.

HTTPSOCKS5SOCKS4aper-connection
SSH · myserver → socks5://127.0.0.1:1080
Git · push → http://127.0.0.1:8080
SFTP · backup → direct (no proxy)

Install

Prerequisite: the Claude Code CLI, installed and logged in once.

1 Claude Code CLI

# install it, then log in once
curl -fsSL https://claude.ai/install.sh | bash
claude

Complete the browser login — the app reuses it, no API key.

2a Debian / Ubuntu (.deb)

sudo dpkg -i "Claude for Linux_*_amd64.deb"

Download the .deb from Releases.

2b Any distro (.AppImage)

chmod +x "Claude for Linux_*_amd64.AppImage"
./"Claude for Linux_*_amd64.AppImage"

Portable — no installation needed.

3 Build from source

npm install
npm run tauri build

Requires Node.js, Rust and the Tauri Linux libraries.
