# Claude برای لینوکس

اپ دسکتاپ اوپن‌سورس و سبک برای لینوکس — یک رابط گرافیکی برای Claude، با دو حالت:

- **گفتگو** — چت مستقیم با Anthropic API (کلید API خودت، استریم زنده، انتخاب مدل Opus/Sonnet/Haiku).
- **کد (Claude Code)** — رابط گرافیکی روی `claude` CLI نصب‌شده؛ یک دستور می‌دهی و روی پوشه‌ی پروژه‌ات کار می‌کند.

ساخته‌شده با **Tauri (Rust) + React + TypeScript**. کلید API فقط روی همین سیستم در `~/.config/claude-linux/settings.json` ذخیره می‌شود و مستقیماً به Anthropic ارسال می‌گردد.

> این پروژه مستقل است و وابسته/تأییدشده توسط Anthropic نیست. آیکون اثر خودِ پروژه است (لوگوی رسمی Claude استفاده نشده).

## اجرا در حالت توسعه

```bash
npm install
npm run tauri dev
```

## ساخت فایل نصبی و نصب

```bash
bash install.sh
```

این اسکریپت کتابخانه‌های سیستمی لازم را (با `sudo`) نصب می‌کند، اپ را می‌سازد، بسته‌ی `.deb` را نصب می‌کند و اجرا می‌کند.

### پیش‌نیازها

- **Node.js** و **Rust** (rustup).
- کتابخانه‌های سیستمی Tauri: `libwebkit2gtk-4.1-dev` و … (اسکریپت نصبشان می‌کند).
- **شبکه برای cargo:** اگر `crates.io` فیلتر است، اسکریپت را پشت پراکسی/VPN اجرا کن:
  `HTTPS_PROXY=http://127.0.0.1:PORT bash install.sh`
